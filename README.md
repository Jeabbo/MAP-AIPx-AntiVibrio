# MAP + AIPx Framework for Anti-Vibrio Peptide Prioritization

This repository contains datasets and analysis scripts associated with the manuscript:

"A Mechanism-Guided Framework for Prioritizing Anti-Vibrio Peptides through Membrane-Interaction Assessment and Ranking"

Contents:
- datasets
- MIC-associated datasets
- MAP scoring workflow
- AIPx scoring workflow
- Statistical analysis scripts
- Figure generation scripts

The repository is provided to support transparency and reproducibility.
